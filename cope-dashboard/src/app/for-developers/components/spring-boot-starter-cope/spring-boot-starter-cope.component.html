Spring Boot Starter

A decision since the inception of COPE was that we would work hard to provide clients with client libraries to make onboarding to COPE as seamless as possible. spring-boot-starter-cope is our first client library and works seamlessly and easily with clients using Spring Boot

Installation

Installation is as easy as adding the maven dependency to your pom.xml file

Configuration

Configuration is also easy. Merely provide your client information, specify your mode and configure your chosen mode with connection information Note: you only need to include the configuration for whichever mode you want to use, changing the value for 'mode' to 'restful' in the sample below will enable synchronous requests using REST.
